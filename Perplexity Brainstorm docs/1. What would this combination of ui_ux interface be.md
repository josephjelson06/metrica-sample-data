# What would this combination of ui/ux interface be called as?

The broad UI/UX category is best called a **multimodal conversational interface** or **multimodal AI assistant**, because it combines voice with on-screen visual interaction and lets the user switch between modes fluidly.[^1][^2][^3]

For your specific product, the most precise term is **multimodal conversational visual analytics interface**, because the coach is not just issuing voice commands — they are asking natural-language questions that drive charts, filters, comparisons, and screen transitions.[^4][^5]

## Best naming options

Depending on how formal or product-oriented you want to be, you could describe it as:[^6][^1][^4]

- **Multimodal conversational interface** — best broad UX term.[^2][^1]
- **Voice-first multimodal interface** — if voice is the primary control method.[^7][^8]
- **Conversational visual analytics system** — best if you want to emphasize data questioning and automatic visual generation.[^5][^4]
- **Multimodal HMI** — more engineering-oriented term, especially if you frame the tablet as a human-machine interface.[^6]
- **Intelligent user interface (IUI)** — if you want to emphasize context-awareness and adaptive behavior.[^2]


## Best fit for your case

For the football tablet you described, I’d call it a **voice-first multimodal conversational visual analytics assistant**.[^1][^4][^5]
That name fits because it includes:

- **voice-first**: coach speaks naturally,[^7]
- **multimodal**: voice + touch + visuals + transitions,[^8][^1]
- **conversational**: the system remembers context and handles follow-up questions,[^9][^4]
- **visual analytics**: it turns football questions into data views and insights.[^4][^5]


## Product-friendly label

If you want a cleaner product/UI label instead of a research-style term, these are good:[^3][^6][^7]

- **AI tactical assistant**
- **Voice-driven analytics dashboard**
- **Conversational coaching dashboard**
- **Interactive match intelligence assistant**

Among these, **conversational coaching dashboard** is easy to understand, but **AI tactical assistant** sounds stronger and more product-ready.[^3][^7]

## Recommended wording

If you are describing it in a case study, product doc, or portfolio, I would use this line:

**“A voice-first multimodal conversational visual analytics assistant for football coaches.”**[^5][^1][^4]
